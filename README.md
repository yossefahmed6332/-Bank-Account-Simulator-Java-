# 💳 Bank Account Simulator - Java

## 📌 Project Description
Bank Account Simulator is a console-based Java application that simulates basic banking operations.  
The project was built to practice Object-Oriented Programming (OOP) concepts such as encapsulation, class design, and data validation.

---

## 🚀 Features

- Create a new bank account
- Generate unique:
  - Account Number
  - User ID
  - PIN
- Deposit money
- Withdraw money (with validation)
- View account balance
- View transaction history
- Change user details:
  - Username
  - Email
  - Password
  - PIN
- Security verification using Password & PIN

---

## 🛠 Technologies Used

- Java
- OOP Principles
- ArrayList
- Console-based interaction (Scanner)

---

## 🧠 OOP Concepts Applied

- Encapsulation
- Separation of concerns
- Class-based structure
- Data validation methods
- Basic security checks
